## 🤖 PHASE 2: DEPLOYING YOUR TELEGRAM BOT

Follow this "Zero-Downtime" deployment workflow:

### 1. Prepare Your Files
Ensure your project folder contains:
* `bot.py` (or `index.js`) - Your main entry file.
* `requirements.txt` (or `package.json`) - Listing all dependencies.
* `.env` - Storing your `BOT_TOKEN` (DO NOT hardcode tokens!).

### 2. Upload to Kerit Panel
1.  Open the **Pterodactyl Panel** provided by Kerit Cloud.
2.  Go to **File Manager** and upload your project files (or use the GitHub Auto-Deploy feature).
3.  In the **Console**, the system will automatically detect your language (Python/Node.js).

### 3. Set Environment Variables
1.  Navigate to the **Settings** or **Variables** tab.
2.  Add a new variable: `BOT_TOKEN`.
3.  Paste your token from @BotFather.

### 4. Ignite 🚀
1.  Go back to the **Console**.
2.  Click the **START** button.
3.  Watch the logs. Once you see `Bot Started Successfully`, you are live 24/7!
